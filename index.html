    Welcome to SANS SEC522!

    SEC522 Lab Workbook Version G02_01

    The goal of the SEC522 Electronic Workbook is to increase the in-class, and, most importantly, after-class value of the course material. This electronic workbook is, and likely always will be, very much a work in progress.

    Contained in the Electronic Workbook, you will find:

    • Electronic Copies of the Lab Guides (copy and paste, FTW!!!)
    • Course Index
    • Tools, cheat sheets, and white papers

    Updating the Electronic Workbook

    Note: To ensure you are seeing the latest version of the Electronic Workbook, please run the following commands in the Terminal.

    cd ~/
sans-update-workbook

    The output should look similar to the following snippet.

    
  f8e4886 (grafted, HEAD, origin/G01_02, G01_02) Lab 3.3: Fixed typo in lab objectives.

    Errata Patches

    On occasion, errata for the VM may be published to update command or lab files. To apply the errata patches, open Firefox inside the virtual machine to view the Electronic Workbook. On the homepage, locate the Errata Patches section Electronic Workbook and click the link to view the patches for the various versions of the VM.

    To find the current version, look on the front page of the Electronic Workbook and locate the “Course Version X##_##”. Click the associated link under the Errata menu to view the patches.

    Course, Workbook, or Lab Bugs

    Please let us know if you find any bugs in the courseware, workbook, or in the labs that we need to squash. Also, reach out if you have suggestions to improve the course (e.g. content/labs/tools that should be added, removed, or updated).
